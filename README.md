
# Simulação Banco (Em Desenvolvimento)

Este é um projeto pessoal para estudo em java. Sendo uma simulação simples de operações como:

        - Consulta de Score;
        - Definição de limite de crédito com base no Score do Usuário;
        - Consulta de limite e saldo disponível em cartão de Crédito/Débito;
        - Cadastro de Usuário com informações básicas necessárias.
    



## Objetivos:

 - Estudo sobre padrão MVC;
 - Integrações com Banco de Dados em SQL (A fazer);
 - Práticas em Interface Gráfica e funcionalidades.
